Terms

Returns a list of terms for which course data is available. Querying http://api.asg.northwestern.edu/terms/?api_key=YOUR_KEY will return a list of objects starting with the most recent term. Each term object will have the following structure:

{% highlight json %} { "term_id": 4560, "name": "2014 Fall", "start_date": "2014-09-23", "end_date": "2014-12-12" } {% endhighlight %}
