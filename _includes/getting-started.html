Getting started

First, log in and apply for a key. Your request should be reviewed within 48 business hours. For now, each project is allotted 10,000 API requests per day. All of your requests to the API must include your key using the key parameter.

The base URL for the Course Data API is:

http://api.asg.northwestern.edu/

To retrieve data from the API, send an HTTP GET request to the base URL + an endpoint. For instance, pasting http://api.asg.northwestern.edu/terms/?key=YOUR_KEY in your browser will return a string in JSON representing a list of terms. (Using a browser extension such as JSONView during development is highly recommended.) Making the same request would look like this using JavaScript with jQuery:

{% highlight javascript %} var params = { key: YOUR_KEY }; $.getJSON('http://api.asg.northwestern.edu/terms/', params, function(data) { // data is the JavaScript array that results from parsing the response console.log(data); }); {% endhighlight %}

Using Python with the requests library:

{% highlight python %} import requests params = { 'key': YOUR_KEY } response = requests.get('http://api.asg.northwestern.edu/terms/', params=params) # calling response.json() returns the response as a Python dictionary or list print response.json() {% endhighlight %}

You can find clients for various languages under the GitHub organization for this project. As of this writing, there are Python and Ruby wrappers available.
